# IOT_Development_Board
Custom IoT development board based on the ESP32-C WROOM module featuring USB-C for power and programming, onboard SD card slot, microphone and light sensor, dedicated OLED display pins, multiple GPIO headers, and integrated reset and boot buttons for easy prototyping and testing.
